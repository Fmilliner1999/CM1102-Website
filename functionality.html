What does C do and why do we use it?

  1. C was (and still is sometimes) the language of choice for operating system development
    • It is used for all of Unix
  2. It allows you direct control over the very low level aspects of the computer
  3. Many legacy programs are written in C
  4. C's pgrograms generally execute and run quickly
  5. C offers flexible memory management allowing the use of different types:
    • Statically, automatically, or dynamically

C could be called a middle level language, as it combines the features of both high and low level languages. This means it can be used for scripting drivers and kernels, and as mentioned earlier, used for operating system development.

It is a widely used and general purpose programming language and can efficiently work on enterprise applications, games, graphic generation and applications requiring calculations.

C implements algorithms and data structures quickly, allowing for faster computation for programs. This enables the use of C in applications like MATLAB and Mathematica, which require a higher degree of calculations.
