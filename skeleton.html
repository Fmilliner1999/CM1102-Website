C is a high-level, general purpose programming language that is ideal for developing firmware or portable applications. Originally intended for writing system software, C was developed at Bell Labs by Dennis Ritchie for the Unix Operating System in the early 70s.

C is a proven, flexible and powerful language and may be used for a variety of different applications. Although high level, C and assembly language share some attributes.

Some of C's most important features include:

  • Fixed number of kewords using control primitives like if, for, while, switch and do while.
  • Multiple logical and mathematical operators, including bit manipulators.
  • Multiple Assignments may be appleid in a single statement.
  • Return values in functions are not always required.
  • Typing is static and all data has type but can be casted.
  • Basic form of modularity, as files may be separately compiled and linked.
